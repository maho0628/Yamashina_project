# ゲームタイトル  
「無人島で生きるには」  
# ゲーム概要  
テキストベースで無人島サバイバルをするシミュレーションゲームです。
無人島には何か一つだけ持ち込めます！  
それを活かしたり、アイテムクラフトや料理をしたりして脱出を目指しましょう。
